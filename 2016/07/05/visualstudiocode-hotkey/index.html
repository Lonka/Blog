Introduction

由於之前常用的 Sublime 多行編輯實在太方便，所以就找了一下 Visual Studio Code 是否有支援相關功能

多行編輯

Shift+Alt+Left

選擇相同詞

Ctrl+D

代碼整理

Shift+Alt+F

代碼開闔

全開|Ctrl+Alt+Shift+]
全瞌|Ctrl+Alt+Shift+[
單開|Ctrl+Shift+]
單闔|Ctrl+Shift+[

快速換行

Alt+↑ or Alt+↓

復製整行

Shift+Alt+↑ or Shift+Alt+↓
