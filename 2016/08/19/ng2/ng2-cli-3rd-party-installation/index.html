Angular2－加入第三方套件（Angular CLI）

發表於 2016-08-19   |   分類於 Angular2   |  

Introduction

原本以為用 Angular2 CLI 建出來的專案加第三方套件也只是用 npm install 後直接於 node_modules 中使用，可是跑起來後一直說找不到 node_modules 資料夾，後來找半天才發現原來專案（angular-cli-build.js）內就有連結教怎麼加入第三方套件

Development

加入 moment.js

moment.js是一個JavaScript的日期處理類庫，用於解析，檢驗，操作，以及顯示日期。

  1. 安裝 monent.js 通過 npm

  2. 加入 moment.js 到angular-cli-build.js的 vendorNpmFiles array

    並重新建置

  3. 設定system-config.ts mappings 知道去那裡找到 moment.js

  4. 匯入和使用在 *.ts

加入 material2

Google Material Design 元件用於 Angular 2 App

  1. 安裝 material2 core 通過 npm

  2. 安裝所需要的 meterial2 package

    目前可用的 package 如下

    • button
    • card
    • checkbox
    • icon
    • input
    • list
    • progress-bar
    • progress-circle
    • radio
    • sidenav
    • toolbar

  3. 加入 material2 到angular-cli-build.js的 vendorNpmFiles array

    並重新建置

  4. 設定system-config.ts mappings 知道去那裡找到 material.js

  5. 匯入和使用在 *.ts

加入 Bootstrap

是最受歡迎的 HTML、CSS 和 JS 框架，用於開發自適應網頁與以行動優先的網站專案。

  1. 加入 bootstrap styles 在index.html

  2. 先安裝 moment.js 之後再安裝 boostrap 通過 npm

  3. 加入 boostrap 到angular-cli-build.js的 vendorNpmFiles array

    並重新建置

  4. 設定system-config.ts mappings 知道去那裡找到 bootstrap.js

  5. 匯入和使用在 *.ts

加入 underscore.js

是一個 JavaScript 工具庫，它提供了一整套函數式編程的實用功能，但是沒有擴展任何 JavaScript 內置對象。

  1. 安裝 underscore 通過 npm

  2. 安裝 typings 至 library

  3. 加入 underscore 到angular-cli-build.js的 vendorNpmFiles array

    並重新建置

  4. 設定system-config.ts mappings 知道去那裡找到 underscore.js

  5. 匯入和使用在 *.ts
    以app.component.ts為例

加入 angular2-in-memory-web-api

可以讓開發人員不用寫後台，直接透過memory的方式，mock 出 WebApi。

  1. 安裝 angular2-in-memory-web-api 通過 npm

  2. 加入 angular2-in-memory-web-api 到angular-cli-build.js的 vendorNpmFiles array

    並重新建置

  3. 設定system-config.ts mappings 知道去那裡找到 angular2-in-memory-web-api

  4. 加入 In Memory Data
    app/in-memory-data.service.ts

  5. 設定 Provide
    RC5 below this
    app.module.ts

    RC4 below this
    main.ts

  6. 透過 http observable 取得資料
    也可以透過 http promise 取得，這裡使用 observable 當範例
    in-memory-data.component.ts

    in-memory-data.component.html

加入 @angular/router

  1. 安裝 @angular/router 通過 npm

  2. 加入 @angular/router 到angular-cli-build.js的 vendorNpmFiles array（理論上應該已經有這一行）

    並重新建置

  3. 設定system-config.ts mappings 知道去那裡找到 @angular/router

  4. 加入 Router 檔
    app.routing.ts
    RC5 below this

    RC4 below this

  5. 設定 Provide
    RC5 below this
    app.module.ts

    RC4 below this
    main.ts

  6. 透過 Router-Outlet 呈現
    app.component.ts

    app.component.html
