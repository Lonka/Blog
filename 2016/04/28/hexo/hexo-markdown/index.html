Markdown 語法

Markdown 是一種輕量級標記式語言，它「使用易讀易寫的純文字格式編寫文件，然後轉換成有效的XHTML(或者HTML)文件」，可以直接用文字檔撰寫，也可以下載編輯器或使用線上編輯器 Cmd Markdown
